Learning Activity 2: JS Basics Program

Part I

1- Create a simple JS line of code that displays “Hello World” on the console.

2- Modify question 1 by adding a comment, variable, and attach it to an HTML element to display the message on browser.

3- Write a simple JS code that prints your full name.

4- Write a simple JS program that compares two values x and y. Assume x = 234 and y = 300. Your program should display a message that indicates the smallest value and the largest value.


4-Write a simple JS program that compares three values x, y, z. You can give any random number for those three variables. The code should display which number has a larger value and which one has a smaller value, among the three.

5- Write a simple JS program that displays all the numbers from 1 to 10 using a for loop.

6- Write a program that display all even numbers from 0 to 10 using a for loop.
